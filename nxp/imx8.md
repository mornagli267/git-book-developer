# i.MX8

SolidRun documentation for **NXP i.MX8** and **i.MX8M** platforms.

Find **i.MX8 SoM**, **System-on-Module**, and **i.MX8 SBC** documentation in one place.

This section covers **i.MX8M Plus**, **i.MX8M Mini**, **i.MX8M Nano**, **i.MX8 Quad**, and **i.MX8 DXL** platforms. It includes **HummingBoard**, **CuBox-M**, hardware user manuals, quick start guides, Linux software, Yocto, U-Boot, and technical articles for embedded systems and edge devices.
