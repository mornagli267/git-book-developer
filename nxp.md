# NXP

### SolidRun NXP platforms

SolidRun products based on **NXP processors**. This section covers **i.MX6**, **i.MX8**, **LX2160A**, and **LX2162A** families.

Use this page to find **SoM**, **COM**, and **SBC / platform** documentation. You’ll also find **hardware user manuals**, **quick start guides**, **Linux**, **Debian**, **Yocto**, **U-Boot**, **DPDK**, and platform resources for **HummingBoard**, **CuBox**, **HoneyComb**, and **ClearFog** systems.
