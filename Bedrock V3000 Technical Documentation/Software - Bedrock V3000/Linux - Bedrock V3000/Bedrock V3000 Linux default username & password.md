# Bedrock V3000 Linux default username & password

## **Bedrock, default Linux credentials**

- root user:

```java
Username: root
Password: root
```

- bedrock user:

```java
Username: bedrock
Password: root
```

> [!IMPORTANT]
> To change the password use
>
> ```java
> sudo passwd <username>
> ```

## Windows RDP (remote desktop connection) credentials

```java
Username: bedrock
Password: root
```
