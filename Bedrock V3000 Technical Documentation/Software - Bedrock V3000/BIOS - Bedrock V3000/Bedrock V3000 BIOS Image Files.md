# Bedrock V3000 BIOS Image Files

> [!NOTE]
> Please follow [BIOS Update procedure](BIOS%20Update%20procedure.md) carefully.
>
> Incorrect BIOS update might lead to BIOS corruption that cannot be recovered and would require an RMA.

> [!IMPORTANT]
> Before updating BIOS, you are advised to [Bedrock V3000 BIOS Settings](Bedrock%20V3000%20BIOS%20Settings.md)
