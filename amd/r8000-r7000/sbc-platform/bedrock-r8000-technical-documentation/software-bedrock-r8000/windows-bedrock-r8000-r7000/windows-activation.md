# Windows Activation

To activate windows you will first need to acquire a Windows activation key.

* Go into the settings menu![](../../../../../.gitbook/assets/image-20231017-063501.png)
* Go to the system tab, and press Activate now at the top part![](../../../../../.gitbook/assets/image-20231017-063757.png)
* press change product key![](../../../../../.gitbook/assets/image-20231017-064044.png)
* Type your Windows activation key and press Next![](../../../../../.gitbook/assets/image-20231017-064204.png)
