# AMD V3000 COM Express changing PCI-e from 2x4 to 1x8

<a id="go-into-bios"></a>

### Go into BIOS

- Turn on the COM Express module
- Repetitively press the DEL / ESC key to enter the BIOS setup

<a id="change-setting"></a>

### Change setting

- Advanced -> AMD PBS -> PCI Express Configurations -> PCIE 1x8 slot = Enable
- save & exit
