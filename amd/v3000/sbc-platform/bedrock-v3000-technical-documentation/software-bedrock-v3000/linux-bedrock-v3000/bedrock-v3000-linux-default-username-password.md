## **Bedrock, default Linux credentials**

- root user:

```
Username: root
Password: root
```

- bedrock user:

```
Username: bedrock
Password: root
```

{% hint style="info" %}
To change the password use
```
sudo passwd <username>
```
{% endhint %}
