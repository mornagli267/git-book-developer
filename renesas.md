# Renesas

### SolidRun Renesas platforms

SolidRun products based on **Renesas RZ processors**. This section covers **RZ/G2L**, **RZ/V2L**, **RZ/G2UL**, **RZ/G2LC**, and **RZ/V2N** families.

Use this page to find **SoM**, **COM**, and **SBC / platform** documentation. You’ll also find **hardware user manuals**, **quick start guides**, **boot options**, **HummingBoard** platform docs, and **embedded AI** resources.
