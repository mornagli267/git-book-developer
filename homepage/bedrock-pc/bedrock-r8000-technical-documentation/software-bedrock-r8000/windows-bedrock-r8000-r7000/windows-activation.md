# Windows Activation

To activate windows you will first need to acquire a Windows activation key.

- Go into the settings menu![](./attachments/image-20231017-063501.png)
- Go to the system tab, and press Activate now at the top part![](./attachments/image-20231017-063757.png)
- press change product key![](./attachments/image-20231017-064044.png)
- Type your Windows activation key and press Next![](./attachments/image-20231017-064204.png)
