# Switching between DisplayPort (USB-C Thunderbolt / Full-size DisplayPort )

To set the display port you will need to open the BIOS menu by booting the device and press the '**Del**' button repetitively until BIOS menu opens.

Go to:

Advance ► AMD PBS ►Display Configuration ►DP2 ► choose your desire Display Port
