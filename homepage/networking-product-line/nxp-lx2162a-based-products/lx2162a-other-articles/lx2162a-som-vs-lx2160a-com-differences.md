# LX2162A SOM vs LX2160A COM Differences

SolidRun offers 2 different product types based on the Layerscape LX2 product family from NXP;

1. [LX2160A COM Express Type 7](https://www.solid-run.com/embedded-networking/nxp-lx2160a-family/lx2162a-som/).
2. [LX2162A SOM](https://www.solid-run.com/embedded-networking/nxp-lx2160a-family/lx2162a-som/).

The purpose of the article is to summarize and identify the main differences between the two products and their different features.

| **Feature** | **LX2160A COM** | **LX2162A SOM** |
| --- | --- | --- |
| CPU Details | 8/12/16 x Cortex A72 up to 2.2GHz | 8/12/16 x Cortex A72 up to 2GHz |
| RAM | Dual channel up to 64GB DDR4 | Single Channel up to 32GB DDR4 |
| RAM type | SO-DIMM | Soldered, onboard |
| eMMC | 64GB | 8GB |
| SerDeses | 24  | 12  |
| Ethernet | 4 x 25 GbE or  <br>1 x 100 GbE\*  <br>4 x 10GbE w/ built in PHY  <br>Sync-E, 1588-V2 | 4 x 25 GbE or  <br>1 x 100 GbE  <br>4 x 10GbE w/ built in PHY  <br>8 x 1GbE\*  <br>Sync-E, 1588-V2 |
| PCIe | 18 (Gen 3.0 – 5 controllers)\* | 8 (Gen 3.0 – 2 controllers)\* |
| SATA | 2 (Gen 3.0)\* | 4 (Gen 3.0)\* |
| Size | 125 x 95 mm | 55 x 48 mm |
| B2B Interface | COM Express Type 7 Headers | 3 x Hirose DF40 connectors |
| Temperature Grade | Commercial: 0°C to 70°C<br><br>Industrial: -40°C to 85°C | Commercial: 0°C to 70°C |

> [!INFO]
> (\*) Configurable SERDESs based on NXP LX2160A processor specifications.
