# HummingBoard Gate/Edge UART console

<a id="j25-connector"></a>

#### J25 Connector

Serial console for the HummingBoard Gate/Edge can be found on connector J25 with GND,TX,RX pins as described in the below picture.

  

![](./attachments/image-20211229-083424.png)

Please reference the board schematics for more information by clicking [here](https://solidrun.atlassian.net/wiki/download/attachments/270401543/HummingBoard2%20Schematics.pdf?api=v2). In order to get serial console you will need a USB to UART (3.3V) , more information can found at [Serial-Console USB->UART](https://solidrun.atlassian.net/wiki/spaces/developer/pages/288096275) .
