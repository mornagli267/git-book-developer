# HummingBoard Gate/Edge UART console

#### J25 Connector

Serial console for the HummingBoard Gate/Edge can be found on connector J25 with GND,TX,RX pins as described in the below picture.

![](../../../../.gitbook/assets/image-20211229-083424.png)

Please reference the board schematics for more information by clicking [here](https://solidrun.atlassian.net/wiki/download/attachments/270401543/HummingBoard2%20Schematics.pdf?api=v2). In order to get serial console you will need a USB to UART (3.3V) , more information can found at [Serial-Console USB->UART](/homepage/iot-industrial-product-line/nxp-imx6-based-products/imx6-other-articles/serial-console-usb-uart.md) .
