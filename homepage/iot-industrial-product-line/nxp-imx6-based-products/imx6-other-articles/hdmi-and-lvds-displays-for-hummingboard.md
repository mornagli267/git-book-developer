# HDMI and LVDS Displays for HummingBoard

<a id="compatible-displays"></a>

#### Compatible Displays

The HummingBoard series supports off the shelf LVDS and HDMI displays by [Powertip](https://www.powertip.com.tw/index.php) and more specifically please use the following [link](https://www.powertip.com.tw/products_2.php?uID=9).

The below table describes the supported part numbers :

|     |     |     |
| --- | --- | --- |
| **Display part number** | **Description** | **Compatible with** |
| PH800480T024-IHC03 | 5” TFT + CTP  (LVDS interface) | HummingBoard Edge and Pro |
| PH800480T013-IHC03 | 7” TFT + CTP  (LVDS interface) | HummingBoard Edge and Pro |
| PH800480T013-IHC08 | 7” TFT + CTP  (HDMI interface) | HummingBoard – whole family (Base / Pro / Gate / Edge) |
