# HummingBoard Edge/Gate Boot Jumpers

{% hint style="warning" %}
Only available for HummingBoard Gate/Edge rev 1.3 and above!
{% endhint %}


On the HummingBoard Gate/Edge it is possible to boot from different media â€“ without blowing the eFuses.

#### Jumpers need to be setup at J5005

![](../../../../.gitbook/assets/image-20211223-194630.png)

To boot from:

* eMMC 3 jumpers will be needed: (1+2),(3+4), (7+8)
* MicroSD 2 jumpers will be needed: (3+4), (5+6)
* M.2 SATA SSD: 1 jumper will be needed: (1+2)
* USB OTG: No jumpers needed: ensure that no jumpers are connected
