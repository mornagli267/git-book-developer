# Windows Activation

To activate windows you will first need to acquire a Windows activation key.

- Go into the settings menu

  ![](../../../../../attachments/bb1eb409-0253-4c27-9169-6697f427e5a9.png)
- Go to the system tab, and press Activate now at the top part

  ![](../../../../../attachments/9f42850e-fea2-4f08-92a5-0f44b2b58c0d.png)
- press change product key

  ![](../../../../../attachments/c03d3dcd-3639-4900-9fbb-01cf14c2a65b.png)
- Type your Windows activation key and press Next

  ![](../../../../../attachments/f6f23b18-404b-40cf-8594-e49a9c693459.png)
