# Cable accessories

## Remote on/off

### Remote power button

A power button on a wire for turning Bedrock on/off when Bedrock is installed in an inaccessible location.

> [!NOTE]
> **Ordering code**: SRBD-CABBTN

![](https://developer.resources.solid-run.com/wiki/download/thumbnails/492437517/1698238205904.jpg?version=1&modificationDate=1698238740973&cacheVersion=1&api=v2&width=760&height=350)

### Harness for custom remote power button

A wire harness for soldering custom power button (optionally with LED) for turning Bedrock on/off when Bedrock is installed in an inaccessible location.

> [!NOTE]
> **Ordering code**: SRBD-CABNOBTN

![](https://developer.resources.solid-run.com/wiki/download/thumbnails/492437517/1698238205889.jpg?version=1&modificationDate=1698238752717&cacheVersion=1&api=v2&width=760&height=350)

## Console mini-USB to USB-A

Console is mini-USB connector with standard USB interface.  
The mini-USB to USB-A allows connecting to a host PC.

> [!NOTE]
> **Ordering code**: SRBD-CABCON

![](https://developer.resources.solid-run.com/wiki/download/thumbnails/492437517/1698238205873.jpg?version=1&modificationDate=1698238762663&cacheVersion=1&api=v2&width=760&height=350)
